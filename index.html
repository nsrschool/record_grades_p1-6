ประมวลจากข้อมูลจริงในระบบทั้งหมด — ผลสัมฤทธิ์ เวลาเรียน 0/ร/มส คุณภาพครู และระบบดูแลช่วยเหลือ

กด "ประมวลผลใหม่" หรือรอสักครู่…

📖 คู่มือการใช้งาน

ขั้นตอนการใช้งานโดยย่อ

  1. ตั้งค่า ข้อมูลสถานศึกษา (โลโก้ ชื่อโรงเรียน ปีการศึกษา ผอ./รองผอ. ลายเซ็น)
  2. เพิ่ม/นำเข้า รายชื่อนักเรียน (ดาวน์โหลดฟอร์มตัวอย่างไปกรอกแล้วนำเข้า)
  3. ตั้ง รายวิชา ตามโครงสร้างหลักสูตร แยกพื้นฐาน/เพิ่มเติม + เวลาเรียน
  4. บันทึกผลการเรียน (Part 3) — กรอกคะแนนรายหน่วย หรือเลือกระดับ 68 โดยตรง
  5. พิมพ์ใบรายงาน (Part 4) — แสดงผลทั้งหลักสูตร 2568 และ 2551 ใน A4 เดียว

การส่งงานและข้อสอบ (ใหม่)

  1. ครูส่งเอกสารได้ 4 ประเภทที่เมนู 📤 ส่งเอกสารการสอน: กำหนดการสอน, แผนการสอน, ข้อสอบกลางปี, ข้อสอบปลายปี
  2. แอดมินกำหนดช่วงเวลาเปิด-ปิดรับงานแต่ละประเภทได้ที่ 🛠️ แอดมิน → 📌 กำหนดงานส่ง (มอบหมายทุกคนหรือรายบุคคล) — นอกช่วงเวลาครูจะส่งไม่ได้ แต่แอดมินส่งแทนได้เสมอ
  3. ถ้ายังไม่กำหนดงานประเภทใด ครูส่งประเภทนั้นได้ตลอดเวลา
  4. ผู้บริหารดูสรุปการส่งได้ที่ 📂 ติดตามเอกสารการสอน (คลิกการ์ดเพื่อดูรายชื่อครูที่ส่งแล้ว/ยังไม่ส่ง)

แดชบอร์ดผู้บริหาร (ใหม่)

  1. เมนู 📊 รายงานสรุป รวมทุกมุมมอง: ✅ การเช็คชื่อ (รายวัน/เดือน/ภาคเรียน/ปีการศึกษา — เห็นเป็นตารางห้อง×วัน เขียว=เช็คแล้ว แดง=ยังไม่เช็ค), 📝 การบันทึกคะแนน (วิชาไหน ครูคนไหน บันทึกแล้วกี่คน/กี่คะแนน), 📤 การส่งงาน, ⚠️ นักเรียนเสี่ยง, 📈 ผลการเรียน
  2. ทุกตารางส่งออกเป็น Excel ได้เพื่อแนบรายงานผู้บริหาร

เอกสารประกอบรายวิชา & นักเรียนเสี่ยง (ใหม่)

  1. เมนู 📚 คำอธิบาย/โครงสร้างวิชา: ครูอัปโหลด คำอธิบายรายวิชา, โครงสร้างรายวิชา, แบบวิเคราะห์ผู้เรียนรายบุคคล ของวิชาที่รับผิดชอบ — ฝังในเล่ม ปพ.5 หน้า "เอกสารประกอบรายวิชา" อัตโนมัติ ส่วนกำหนดการสอนรายภาคดึงจากเมนูส่งเอกสารการสอนมาแสดงรวมให้ ไม่ต้องอัปซ้ำ
  2. เมนู 🔍 นิเทศ/เยี่ยมชั้นเรียน: ผู้บริหาร/แอดมินบันทึกผลนิเทศรายครู (คะแนน 0-100 → ระดับคุณภาพอัตโนมัติ) มีสรุปรายครู พิมพ์แบบบันทึกรายครั้ง และ Export Excel — ครูเห็นเฉพาะบันทึกของตนเอง
  3. เมนู ⚠️ นักเรียนเสี่ยง: กด 📄 บันทึก / ✉️ เชิญ ท้ายรายชื่อ ระบบสร้าง บันทึกข้อความรายงานผู้บริหาร และ หนังสือเชิญผู้ปกครอง เติมข้อมูลนักเรียนให้อัตโนมัติ พร้อมพิมพ์ทันที (พิมพ์ทีเดียวทุกคนได้) — แอดมินแก้ข้อความแบบฟอร์มได้ที่ 🛠️ แอดมิน → 📝 ฟอร์มนักเรียนเสี่ยง

ประวัตินักเรียน & จำหน่าย & ตารางสอน (ใหม่)

  1. 🪪 ประวัตินักเรียน: แอดมินนำเข้าไฟล์ DMC (.xlsx) เติมข้อมูลหลักอัตโนมัติ • ครูประจำชั้นกรอกรูปถ่าย (≤5MB ระบบย่อเหลือ 600px) ชื่อเล่น เบอร์ผู้ปกครอง ฯลฯ • พิมพ์ระเบียนประวัติและ 🪪 บัตรนักเรียนทั้งห้องได้ (8 ใบ/A4)
  2. ลบนักเรียน (แอดมิน) = จำหน่ายไปเก็บที่เมนู 🗃️ นักเรียนจำหน่าย — คะแนน/เช็คชื่อ/เกรดไม่หาย ดูย้อนหลัง พิมพ์รายงาน และกู้คืนได้ • เลขที่ในห้องขยับให้อัตโนมัติพร้อมพรีวิวก่อนยืนยัน
  3. ✏️ แก้ไขตารางสอน (แอดมิน): 🛠️ แอดมิน → ตารางสอน → เลือกครู → คลิกคาบเพื่อแก้/ย้าย/ลบ หรือ ➕ เพิ่มคาบ ระบบเตือนการชนอัตโนมัติ • เมนู 🗓️ ตารางสอน เลือกดูรายครู/รายห้อง และ 🖨️ พิมพ์ A4 แนวนอนได้
  4. 🧑‍🎓 สถิตินักเรียน: 📊 รายงานสรุป → แท็บสถิตินักเรียน (ชาย/หญิงรายห้อง ศาสนา กรุ๊ปเลือด ความด้อยโอกาส + Excel)

วิธี Deploy เป็น Web App (สำหรับครูที่นำโค้ดไปใช้ต่อ)

  1. สร้าง Google Sheet ใหม่ → Extensions → Apps Script → วาง Code.gs
  2. รันฟังก์ชัน INIT 1 ครั้งเพื่อให้สิทธิ์และสร้างชีต
  3. Deploy → New deployment → Web app → Execute as: Me → Who has access: Anyone
  4. คัดลอก Web App URL มาวางในไฟล์ index.html ที่ตัวแปร GAS.URL
  5. อัปโหลด index.html ขึ้น GitHub Pages
