📖 คู่มือการใช้งาน

1. หน้าแรก — ดูภาพรวมจำนวนนักเรียนและสรุปรายชั้น
2. รายงานสรุป — ค้นหา/กรองตามชั้น สั่งพิมพ์ หรือส่งออกเป็น Excel
3. ตั้งค่าระบบ — กรอกข้อมูลสถานศึกษา อัปโหลดโลโก้/ลายเซ็น และจัดการรายชื่อนักเรียน (เพิ่ม/แก้ไข/ลบ/นำเข้าไฟล์)
4. สำรอง/นำเข้า/ล้างข้อมูล — ปุ่มด้านบนขวา สำหรับสำรองเป็นไฟล์ .json หรือคืนค่าจากไฟล์สำรอง

⚙️ วิธี Deploy เป็น Web App (สำหรับครูที่นำโค้ดไปใช้)

1. สร้าง Google Sheet ใหม่ → เมนู ส่วนขยาย (Extensions) → Apps Script
2. วางโค้ด Code.gs และสร้างไฟล์ HTML ชื่อ index แล้ววางโค้ด HTML
3. กด Deploy → New deployment → Web app • Execute as: Me • Who has access: Anyone
4. คัดลอก Web App URL ไปเปิดใช้งาน
💡 หากเพิ่มฟังก์ชันใหม่ ต้อง Deploy → Manage deployments → ✏️ → New version → Deploy ทุกครั้ง
