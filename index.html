📖 คู่มือการใช้งาน

ขั้นตอนการใช้งานโดยย่อ

  1. ตั้งค่า ข้อมูลสถานศึกษา (โลโก้ ชื่อโรงเรียน ปีการศึกษา ผอ./รองผอ. ลายเซ็น)
  2. เพิ่ม/นำเข้า รายชื่อนักเรียน (ดาวน์โหลดฟอร์มตัวอย่างไปกรอกแล้วนำเข้า)
  3. ตั้ง รายวิชา ตามโครงสร้างหลักสูตร แยกพื้นฐาน/เพิ่มเติม + เวลาเรียน
  4. บันทึกผลการเรียน (Part 3) — กรอกคะแนนรายหน่วย หรือเลือกระดับ 68 โดยตรง
  5. พิมพ์ใบรายงาน (Part 4) — แสดงผลทั้งหลักสูตร 2568 และ 2551 ใน A4 เดียว

วิธี Deploy เป็น Web App (สำหรับครูที่นำโค้ดไปใช้ต่อ)

  1. สร้าง Google Sheet ใหม่ → Extensions → Apps Script → วาง Code.gs
  2. รันฟังก์ชัน INIT 1 ครั้งเพื่อให้สิทธิ์และสร้างชีต
  3. Deploy → New deployment → Web app → Execute as: Me → Who has access: Anyone
  4. คัดลอก Web App URL มาวางในไฟล์ index.html ที่ตัวแปร GAS.URL
  5. อัปโหลด index.html ขึ้น GitHub Pages
