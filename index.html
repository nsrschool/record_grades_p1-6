📈 สรุปเวลาเรียนรายภาค

⚠️ นักเรียนที่เวลาเรียนต่ำกว่าเกณฑ์ (เสี่ยง "มส" — ไม่มีสิทธิ์เข้ารับการประเมินปลายภาค) จะแสดงสีแดง

นำเข้าตารางสอนทั้งโรงเรียนได้ที่เมนู 🛠️ แอดมิน

ยังไม่มีข้อมูลตารางสอน

🔧 รายชื่อนักเรียนติด ร / 0 / มส — ภาคเรียนที่

ตามแนวปฏิบัติการวัดและประเมินผลฯ 2551: การสอบแก้ตัวให้ได้ระดับผลการเรียนไม่เกิน "1" | ระบบจะเก็บประวัติผลเดิมและวันที่แก้ไว้ให้

ห้องเลขที่ชื่อ-นามสกุลรายวิชาผลปัจจุบันประวัติการแก้ดำเนินการ

🎉 ไม่มีนักเรียนติด ร/0/มส ในภาคเรียนนี้

เลือกนักเรียนแล้วกดสร้างใบรายงาน

⚠️ นักเรียนกลุ่มเสี่ยง ภาคเรียนที่

ห้องเลขที่ชื่อ-นามสกุลGPAปัญหาที่พบ

🎉 ไม่พบนักเรียนกลุ่มเสี่ยง

👩‍🏫 บัญชีครู

รองรับไฟล์ "ตารางครู" ของโรงเรียน (ชีตสรุปภาระงานสอน + ชีตรายครู) — ระบบจะสร้างบัญชีครูใหม่อัตโนมัติ (PIN เริ่มต้น 1234, ข้ามคนที่มีแล้ว) และแทนที่ตารางสอนเดิมทั้งหมด

ชื่อPINสิทธิ์ประจำชั้น

🔒 ล็อก/อนุมัติผลการเรียน (ราย ห้อง + ภาคเรียน)

เมื่อล็อกแล้ว ครูจะแก้คะแนน/เกรด/คุณลักษณะ/กิจกรรมของห้องนั้นในภาคเรียนนั้นไม่ได้

🧑‍🎓 จัดการรายชื่อนักเรียน

⚠️ การนำเข้าไฟล์จะลบรายชื่อเดิมทั้งหมดแล้วแทนที่ด้วยรายชื่อใหม่ | ฟอร์ม: ชั้น, เลขที่, เลขประจำตัว, คำนำหน้า, ชื่อ, นามสกุล | หรือใช้เมนู NSR → 📋/📥 ในหน้า Google Sheets ก็ได้

เลขประจำตัวเลขที่ชื่อ-นามสกุลห้อง

🎓 ประวัติ GPA ภาคเรียนก่อนหน้า (สำหรับ GPAX สะสม)

ใช้กรอกผลภาคเรียนเก่า (เช่น ม.1-ม.2 ของเด็ก ม.3) — กรอก ผลรวมแต้ม (เกรด×หน่วยกิต) และ หน่วยกิตรวม ของภาคนั้น ระบบจะนำไปรวมคำนวณ GPAX ให้ถูกต้อง

ปีภาคผลรวมแต้มนก.GPA

🎚️ ตารางตัดเกรด 8 ระดับ (แก้ช่วงคะแนนได้)

คะแนนต่ำสุดคะแนนสูงสุดเกรดความหมาย

📖 คู่มือการใช้งานโดยย่อ

ลำดับการเริ่มใช้งาน (แอดมิน): ⚙️ ตั้งค่า (ข้อมูลโรงเรียน, ภาคเรียน, สัดส่วนกลาง, ช่วงวันที่ภาคเรียน) → 🛠️ แอดมิน (เพิ่มบัญชีครู + นำเข้ารายชื่อนักเรียน) → 🏫 รายวิชา (เพิ่มวิชาของแต่ละห้อง/ภาค หรือใช้ปุ่มคัดลอกไปอีกห้อง)

งานประจำของครู: 📝 บันทึกคะแนน — สร้างช่องคะแนน (ระหว่างภาค/ปลายภาค) แล้วกรอกคะแนน ระบบตัดเกรด 8 ระดับให้อัตโนมัติตามสัดส่วนของวิชา | กรณีพิเศษกดตั้งผล "ร" หรือ "มส" รายคนได้ | 🎓 คุณลักษณะฯ และ 🎯 กิจกรรม กรอกเป็นตารางทั้งห้อง | ✅ เช็คชื่อรายวัน — ระบบสรุปเวลาเรียนรายภาคและเตือนเด็กเสี่ยง "มส" ให้

ปิดภาคเรียน: 🔧 แก้ ร/0/มส (ระบบเก็บประวัติการแก้ + เตือนเกณฑ์แก้ตัวไม่เกิน 1) → 📋 สรุปผล/Export กดส่งออกไฟล์ .xlsx รายห้อง รายภาค สำหรับนำเข้า SchoolMIS → แอดมินกด 🔒 ล็อกผล ราย ห้อง+ภาค → 🖨️ พิมพ์ใบรายงานรายคน หรือใบสรุปสะสม GPAX (ของ ม.3 ให้กรอกผลภาคเก่าที่เมนูแอดมิน → ประวัติ GPA ก่อน)

🚀 วิธี Deploy เป็น Web App (สำหรับครูที่นำไปใช้ต่อ)

  1. สร้าง Google Sheet ใหม่ → เมนู Extensions → Apps Script
  2. วางโค้ดจากไฟล์ Code.gs ทับโค้ดเดิมทั้งหมด → กด 💾 บันทึก
  3. รันฟังก์ชัน INIT หนึ่งครั้ง → กดอนุญาตสิทธิ์ (Authorize) ให้ครบ — ระบบจะสร้างชีตทุกแผ่นอัตโนมัติ
  4. กด Deploy → New deployment → ⚙️ เลือก Web app → Execute as: Me → Who has access: Anyone → Deploy
  5. คัดลอก Web App URL มาวางในไฟล์ index.html บรรทัดบนสุด (ตัวแปร API_URL) → อัปโหลด index.html ขึ้น GitHub Pages
  6. เพิ่มบัญชีครูคนแรก: เปิดชีต Teachers ใส่แถว เช่น T1 | admin | 1234 | admin | แล้วล็อกอินด้วย admin/1234 เพื่อตั้งค่าระบบต่อ
  7. ⚠️ ทุกครั้งที่แก้ Code.gs ต้อง Deploy → Manage deployments → ✏️ Edit → New version → Deploy | แก้ index.html ต้องอัปขึ้น GitHub ใหม่ + hard refresh
